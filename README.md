# ❓ QUIZ-APP（React製クイズアプリ）

## 🔍 概要
このアプリは、Reactを用いて作成されたシンプルなクイズアプリです。  
画面に表示される選択式の質問に答え、全4問のスコアを確認することができます。  
リアルタイムでの解答処理、スコア計算、正誤判定を通じて、Reactの基礎的なステート管理やイベント処理を学ぶことを目的としています。

---

## 🛠 使用技術
- React（CRA）
- JavaScript (ES6+)
- HTML / CSS
- Git / GitHub

---

## ✨ 主な機能
- クイズ形式の選択問題を出題
- 選択肢をクリックすると即座に正誤を判定
- 正解数に応じたスコアを表示
- スコアボードで解答履歴の確認が可能
- 正解は緑、間違いは赤でハイライト表示
